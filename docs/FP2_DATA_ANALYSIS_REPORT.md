## 🔍 ВЫЯВЛЕННЫЕ ПРОБЛЕМЫ

### Проблема 1: Координаты обновляются редко

**Симптомы:**
- Movement Event = 7 (Moving)
- Координаты не меняются 30+ секунд
- Delta = +0, +0

**Причина:**
- Ограничение Aqara Cloud API (EU регион)
- Координаты обновляются только при макро-движениях (>30-50 см)

**Решение:**
- Использовать Movement Event как индикатор движения
- UI уже показывает "LIVE" при Movement Event codes 1,2,3,4,7

---

### Проблема 2: Зонные endpoint'ы могут отсутствовать

**Симптомы:**
- Нет `3.*.85` кроме `3.51.85`
- Нет `13.12*.85` кроме `13.120.85`

**Причина:**
- Зоны не настроены в Aqara Home app
- Или используется режим без зонирования

**Решение:**
- Настроить зоны в Aqara Home app
- Проверить настройки FP2
