## 💾 АВТОСОХРАНЕНИЕ

### Как работает:

1. Выбираете язык (RU или EN)
2. Сохраняется в `localStorage` браузера
3. При следующем открытии - тот же язык
4. Не нужно настраивать заново

**Где хранится:**
```javascript
localStorage.getItem('fp2_lang') // 'ru' или 'en'
```

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Реализация:

**i18n система:**
```javascript
const i18n = {
  en: { /* 15+ переводов */ },
  ru: { /* 15+ переводов */ }
};

app.applyLanguage('ru'); // Переключить на русский
```

**HTML элементы:**
```html
<button data-i18n="tab_dashboard">Панель управления</button>
```

**При переключении:**
- Находит все `[data-i18n]` атрибуты
- Берёт перевод из словаря
- Заменяет текст элементов
