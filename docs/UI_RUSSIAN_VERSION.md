## 💾 СОХРАНЕНИЕ НАСТРОЕК

### Автоматическое сохранение

Выбранный язык **автоматически сохраняется** в localStorage браузера.

**При следующем открытии:**
- Интерфейс загрузится на последнем выбранном языке
- Не нужно переключать заново

**Где хранится:**
```javascript
localStorage.setItem('fp2_lang', 'ru'); // или 'en'
```

## 🎯 КАК ЭТО РАБОТАЕТ

### Архитектура i18n системы

```javascript
// Словарь переводов
const i18n = {
  en: {
    subtitle: 'Complete telemetry from Aqara FP2 sensor...',
    tab_dashboard: 'Dashboard',
    // ... другие переводы
  },
  ru: {
    subtitle: 'Полная телеметрия с сенсора Aqara FP2...',
    tab_dashboard: 'Панель управления',
    // ... другие переводы
  }
};

// Применение языка
app.applyLanguage('ru'); // Переключает на русский
app.applyLanguage('en'); // Switches to English
```

### Элементы с data-i18n

HTML элементы используют атрибут `data-i18n` для автоматического перевода:

```html
<p class="subtitle" data-i18n="subtitle">
  Полная телеметрия с сенсора Aqara FP2...
</p>

<button class="nav-tab" data-tab="dashboard" data-i18n="tab_dashboard">
  Панель управления
</button>
```

**При переключении языка:**
1. Система находит все элементы с `data-i18n`
2. Берёт значение атрибута (например, `"tab_dashboard"`)
3. Ищет перевод в словаре `i18n[lang]["tab_dashboard"]`
4. Заменяет текст элемента на переведённое значение

## 🔧 ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ

### Файлы с переводами

**app.js:**
```javascript
const i18n = {
  en: { ... },  // 15+ переводов
  ru: { ... }   // 15+ переводов
};
```

**Методы:**
- `applyLanguage(lang)` - Применяет язык ко всем элементам
- `setupLanguageToggle()` - Настраивает кнопку переключения
- Сохраняет выбор в `localStorage`

### Расширение переводов

Для добавления новых переводов:

1. Добавьте ключ в оба словаря (`en` и `ru`):
```javascript
const i18n = {
  en: {
    // ... существующие
    new_key: 'English translation'
  },
  ru: {
    // ... существующие
    new_key: 'Русский перевод'
  }
};
```

2. Добавьте `data-i18n` атрибут в HTML:
```html
<span data-i18n="new_key">Текст</span>
```

3. Готово! Перевод применится автоматически.
