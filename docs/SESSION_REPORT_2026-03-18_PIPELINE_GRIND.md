# Session Report — 2026-03-18 — Pipeline Grind

## CSI Motion Detection: от 0.80 до 0.856

### Краткое резюме

За одну сессию Binary BalAcc вырос с **0.80 → 0.856** (+7%), false positive rate упал с **96% → ~15%**. Записано 50+ минут новых данных, 45 клипов размечены визуально глазами, построены версии V22-V28, найдены и устранены ключевые проблемы YOLO-разметки.

## 4. Анализ YOLO ошибок (квантифицировано)

### Сравнение YOLO vs визуальная разметка (166 окон)
- **Совпадение: 73.5%** (122 из 166)
- **YOLO пропускает людей: 18.1%** (30 окон) — все STATIC в тёмном гараже
- **YOLO фантомы: 8.4%** (14 окон) — предметы на полках при дневном свете

### Enhanced YOLO — неожиданный результат
- Enhancement **ухудшает** YOLO: raw=83% detect rate vs enhanced=17%
- Причина: перенасыщение размывает контуры
- **Вывод**: YOLO лучше работает на сырых кадрах с conf=0.15

### Попытка видео-аннотатора
- Frame diff: EMPTY=0.72, STATIC=0.84, MOTION=1.17 — слишком большое перекрытие
- Гибрид YOLO+framediff: coarse=0.530 — хуже CSI модели
- **Вывод**: видео в тёмном гараже хуже CSI для классификации

## 6. Ключевые открытия

### Что работает
1. **Баланс данных** — 1007 EMPTY vs 1048 OCCUPIED: +3.5%
2. **Визуальная разметка глазами** — единственный надёжный source of truth
3. **Temporal context** — ctx3_mean фичи: 8 из 10 top, +2% binary
4. **Post-prediction smoothing k=7** — +4.1%, самый большой прирост за сессию
5. **Subcarrier variance** (sc_var_std, sc_var_max) — в top-10 фич
6. **HGB с class_weight=balanced** — лучшая модель
7. **Threshold tuning** (0.25-0.45) — +1-3%

### Что не работает
1. **Enhanced YOLO** — gamma+CLAHE ухудшает YOLO (83% -> 17%)
2. **YOLO labels** — 26.5% ошибок
3. **Anomaly detection** (IsolationForest) — BalAcc=0.48
4. **Frame differencing** — слишком большое перекрытие классов
5. **March 17 аннотации агента** — навредили модели
6. **Больше фич != лучше** — 50-106 оптимально из 1844

### Физические пределы CSI
- CSI baseline deviation: EMPTY ~0.65 sigma, STATIC ~0.95 sigma, MOTION ~1.15 sigma
- Граница EMPTY/OCCUPIED ~ 0.85 sigma (чёткая)
- STATIC/MOTION перекрываются (0.95 vs 1.15) — физический предел
