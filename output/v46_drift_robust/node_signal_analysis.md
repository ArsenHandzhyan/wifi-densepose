### 9.2 Рекомендации для V46 модели

1. **Использовать межнодовые ratio** вместо абсолютных амплитуд — дрифт составляет десятки процентов
2. **CV (std/mean)** и **IQR/median** — наиболее дрифт-устойчивые фичи
3. **Временная вариация** (std mean_amp по пакетам в окне) — ключевой признак motion vs static
4. **Приоритетные пары** для межнодовых ratio — топ-5 из таблицы 6.1
5. **Взвешивание нод**: больший вес для n06(.132) (presence) и n01(.137) (motion)
6. **RSSI-нормализация**: частично компенсирует дрифт, особенно для дальних нод
7. **Субнесущие**: спектральные ratio (low/mid/high) стабильнее абсолютных значений
