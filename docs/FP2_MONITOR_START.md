## 📋 Что делать:

### 1️⃣ Запустите мониторинг

Скопируйте и выполните команду выше в терминале

### 2️⃣ Дождитесь первого замера

Увидите примерно:
```
🔍 МОНИТОР ВСЕХ ENDPOINT'ОВ FP2
====================================================
✅ Найдено: lumi1.54ef4479e003 (Датчик присутствия FP2)

================================================================================
📊 Замер #1 @ 02:30:15.123
================================================================================

⏸️  Нет изменений
```

### 3️⃣ Начните тестирование

#### Тест 1: Встаньте перед датчиком
```
Ожидайте увидеть:
⚡ ИЗМЕНИЛИСЬ РЕСУРСЫ (2):
  🔴 ВАЖНО 3.51.85: 1 (Присутствие (0/1))
  🔴 ВАЖНО 13.27.85: 1 (Код события движения)
```

#### Тест 2: Помашите руками
```
Ожидайте увидеть:
⚡ ИЗМЕНИЛИСЬ РЕСУРСЫ (1):
  🔴 ВАЖНО 13.27.85: 2 (Код события движения)
```

#### Тест 3: Походите из стороны в сторону
```
Ожидайте увидеть:
⚡ ИЗМЕНИЛИСЬ РЕСУРСЫ (3):
  🔴 ВАЖНО 13.27.85: 7 (Код события движения)
  🔴 ВАЖНО 4.22.700: 2 цель(ей): [(52, 335), (23, 315)]
      Сырые: [{"rangeId":0,"x":52,"y":335,"state":"1"}, ...]
  🟡 ЗОНА 3.1.85: Зона 1: ЗАНЯТА
```

#### Тест 4: Выйдите из комнаты
```
Ожидайте увидеть:
⚡ ИЗМЕНИЛИСЬ РЕСУРСЫ (2):
  🔴 ВАЖНО 3.51.85: 0 (Присутствие (0/1))
  🔴 ВАЖНО 13.27.85: 6 (Код события движения)
```

---

## 🎯 Расшифровка маркеров:

- **🔴 ВАЖНО** - Критические изменения (присутствие, движение, координаты)
- **🟡 ЗОНА** - Изменения в зонах (занятость, количество целей)
- **⚪** - Остальные данные (RSSI, свет, угол)

---

## ⌨️ Управление:

- **Ctrl+C** - Остановить мониторинг
- **Ctrl+Z** - Поставить на паузу (затем `fg` чтобы продолжить)
